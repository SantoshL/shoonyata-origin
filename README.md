# Shoonyata Origin

An interactive exploration of observer-dependent meaning and the unreachable origin (Shoonyata).

## Concept

- Point → Line → Plane → 3D Morphing Wireframe
- Delta CMH: measures distance from the origin (always approximate)
- Multi-observer consensus mode (future feature)

## Tech Stack

- HTML/CSS/JavaScript
- Three.js for 3D rendering
- Deployed on Vercel

## Local Development

1. Open `index.html` in a browser
2. Or use Live Server extension in VSCode

## Deployment

Connected to GitHub → auto-deploys to Vercel on push.
